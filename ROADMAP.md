# iOSAppTemplates Roadmap

Last updated: 2026-04-01

## Goal

Make iOSAppTemplates the default world-class starting point for premium SwiftUI apps.

## Current Gaps

- first-time users still have too many docs to choose from
- discoverability can improve with a clearer gallery and stronger onboarding path
- generator proof and benchmark visibility should be easier to find
- community growth can be helped by a more obvious contribution path

## Sprint 1: Onboarding And Showcase

- add one obvious getting-started path for first-time users
- add a gallery surface that maps templates to use cases
- make benchmark, architecture, and generator docs easier to reach from the README
- improve README discoverability with stronger internal linking

## Sprint 2: Generator Hardening

- improve generator documentation and tests
- clarify generator API boundaries
- document advanced customization and migration patterns

## Sprint 3: Product Proof

- surface real-world apps or simulator demos
- publish benchmark baselines per template family
- make accessibility and platform support easier to verify
