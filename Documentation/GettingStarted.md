## Recommended First 20 Minutes

1. Read the root `README.md`.
2. Run the generator in interactive mode.
3. Pick one template that matches your product idea.
4. Read the matching guide inside `Documentation/`.
5. Open `Benchmarks/README.md` to see the quality bar.

## Suggested First Sprint

- generate one template
- rename the app and bundle identifiers
- replace mock data with your domain model
- run tests
- review architecture and customization guides before large edits
